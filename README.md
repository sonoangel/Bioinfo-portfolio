# Bioinformatics analysis of 20 human TP53 mRNA sequences
downloaded from NCBI using Python and Biopython.

## What this project does
- Downloads TP53 sequences from NCBI via Entrez API
- Calculates nucleotide composition and GC content
- Generates 3 visualizations
- Exports results to CSV

## Key findings
- Average sequence length: ~2018 nucleotides
- Average GC content: ~53.9%
- Balanced nucleotide distribution across sequences

## Technologies
Python · Biopython · Pandas · Matplotlib · Seaborn

## How to run
pip install biopython pandas matplotlib seaborn
jupyter notebook analysis_tp53.ipynb
